Name

acosh — return the arc hyperbolic cosine of the parameter

Declaration

genType acosh( genType x);
 

Parameters

x

Specify the value whose arc hyperbolic cosine to return.

Description

acosh returns the arc hyperbolic cosine of x; the non-negative inverse of cosh. Results are undefined if x<1.
