Name

dot — calculate the dot product of two vectors

Declaration

float dot( genType x,
  genType y);
 

Parameters

x

Specifies the first of two vectors

y

Specifies the second of two vectors

Description

dot returns the dot product of two vectors, x and y. i.e., x [ 0 ] ⋅ y [ 0 ] + x [ 1 ] ⋅ y [ 1 ] + ...
