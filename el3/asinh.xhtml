Name

asinh — return the arc hyperbolic sine of the parameter

Declaration

genType asinh( genType x);
 

Parameters

x

Specify the value whose arc hyperbolic sine to return.

Description

asinh returns the arc hyperbolic sine of x; the inverse of sinh.
